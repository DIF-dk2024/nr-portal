# Портал Покупателей — Numerical Research

Flask-приложение для публикации заявок покупателей и сбора предложений от агентов.

## Основной функционал

- Главная страница: **Портал Покупателей**.
- Раздел: **Заявки на покупку**.
- Публичная кнопка **«Хочу Купить»** создаёт новую заявку без входа в админку.
- Заявки создаются без загрузки файлов и фото.
- В каждой заявке агенты могут оставить **Предложение Агента**.
- Для отправки предложения обязательны:
  - `Мой контакт`
  - `Мой Пароль`
  - текст предложения
- Видимое значение пароля в форме по умолчанию: `Free`.
- Реальное секретное слово хранится на бэкенде в файле:

```text
data/agent_offer_password.txt
```

Если изменить содержимое этого файла на Render Disk, видимое значение `Free` на странице не изменится. Отправить предложение сможет только тот, кто знает актуальное секретное слово.

## Render

Переменные среды:

```text
SECRET_KEY=your-long-random-secret
ADMIN_KEY=your-admin-password
DATA_DIR=/var/data/data
UPLOADS_DIR=/var/data/uploads
```

`UPLOADS_DIR` оставлен для совместимости со старыми карточками, но новые заявки файлы не загружают.
